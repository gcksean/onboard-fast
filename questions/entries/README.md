# Question Entry Template

```yaml
---
id: q-<slug>
title: <question title>
status: active
tags: []
updated: YYYY-MM-DD
source: manual
asked_by: self
answer_status: open|partially_answered|answered
answered_on:
linked_updates: []
---
```

## Resolution Rules
- If answered, summarize the answer in the question entry.
- Add links to updated entries in `resources`, `concepts`, and/or `procedures`.
- Keep historical context but avoid stale ambiguity.
